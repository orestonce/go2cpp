go2cpp : 将go接口方便导出为c++接口的一个工具
====

支持的函数:
  * 入参个数无限制
  * 出参个数为0或者1

支持的类型(golang):
  * [x] bool
  * [x] int8/uint8(byte)
  * [ ] int16/uint16
  * [x] int32/uint32
  * [ ] int64/uint64
  * [x] int
  * [x] string
  * [x] slice/切片
  * [x] struct
  * [x] map
  * [x] float32/float64
